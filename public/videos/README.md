# Vídeo da abertura

A primeira seção da página usa estes arquivos:

- `hero-sgs-clean.mp4`: vídeo exibido em reprodução automática, sem áudio e em loop.
- `hero-poster-clean.webp`: imagem mostrada durante o carregamento ou quando o visitante prefere reduzir movimento.

Os arquivos sem o sufixo `-clean` preservam a versão anterior como referência visual.

Para trocar o vídeo ativo, substitua `hero-sgs-clean.mp4` mantendo o mesmo nome. Prefira MP4 com codec H.264, proporção 16:9, resolução de até 1920 × 1080 e arquivo otimizado para web.
